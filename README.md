# Spotify Artist Popularity Analysis (2014–2024)

This data science project analyzes artist popularity using a Spotify long tracks dataset.

## Objective
To identify the most popular artists based on:
- Number of long tracks
- Total duration of tracks
- Consistency across years

## Tools Used
- Python
- Pandas
- Matplotlib
- Google Colab

## Dataset
Spotify Long Tracks Dataset (2014–2024)

## Key Analysis
- Top artists by track count
- Artists with longest total duration
- Most consistent artists over years

## Outcome
This project demonstrates how data analysis can be used to extract meaningful insights about artist dominance in long-format music.
